# daily-proxy

## Install
```bash
$ git clone https://github.com/ncysatnaf/daily-proxy
```

## Usage
#### Run Server
```bash
$ node index.js -p 3001
```

#### Send request
```js
fetch('http://127.0.0.1:3001?https://www.artstation.com/search/users.json?q=m4')
```
